REST API with Node.js, Express & PostgreSQL

------------ RESTAURANTS ----------------------

// GET
/restaurants --> all restaurants
/restaurant/:id --> specific restaurant

// POST
/add/restaurant --> add restaurant

// DELETE
/delete/restaurant/:id --> delete restaurant
